# GoTrip

GoTrip is a full-stack travel and tourism platform built with PHP, MySQL, HTML, CSS, and JavaScript.

It integrates Google Maps services to provide personalized destination discovery, nearby food and tourist attraction recommendations, budget and area-based filtering, travel quizzes, user reviews, image uploads, and an admin dashboard.

Users can explore destinations based on location, preferences, and pricing while receiving smart travel recommendations for a more personalized travel experience.

## Features

- Destination discovery
- Google Maps integration
- Nearby food recommendations
- Tourist attraction recommendations
- Budget-based filtering
- Area-based filtering
- Travel quizzes
- User reviews and ratings
- Image upload support
- Admin dashboard
- Responsive interface

## Tech Stack

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Google Maps API

## Setup

1. Clone the repository
2. Import `setup_full.sql` into MySQL
3. Configure database credentials in `db.php`
4. Start Apache and MySQL
5. Open the project in your browser
